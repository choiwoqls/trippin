**여행 일정 관리 웹사이트**

**진행기간**
2023.08 ~ 2023.09

**참여인원**
front-end 1명
back-end 5명

**담당기능**
- 여행 일정 관리(여행 생성, 수정, 삭제)
- 관광지 필터 검색(지역, 유형<숙소,맛집,관광지>, 성격<호텔,모텔,술집,카페,체험,명소>)
- 여행 후기 관리(후기 생성, 삭제, 별점, 사진, 한줄평)
- 후기 블로그(게시글 좋아요, 여행 일정 복사, 댓글)


**사용 기술**
 - Spring framework legacy project
 - Java 11
 - OracleSQL
 - JSP, HTML, CSS, Ajax, JavaScript, jQuery
